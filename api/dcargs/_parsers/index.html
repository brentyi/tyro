dcargs._parsers.is_possibly_nested_type(typ: Any) → bool[source]#

Heuristics for determining whether a type can be treated as a ‘nested type’, where a single field has multiple corresponding arguments (eg for nested dataclasses or classes).

Examples of when we return False: int, str, List[int], List[str], pathlib.Path, etc.
